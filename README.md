line-comment
============
### Description

An vim plugin, comment single line code

一個單行注釋代碼的小插件.

目前只支持以下幾種語言 

`vim script` `php` `python` `css` `c` `html` `xml`

### Install

* 拷貝 `line-comment.vim` 到~/.vim/plugin/, 重啟vim即可.
* 或在clone 當前版本庫, 編輯.vimrc 文件裏. `source clone-dir/line-comment.vim`
